# Calci

A simple desktop calculator built using Python and CustomTkinter.

---

## Features

- Basic arithmetic operations
- Right-aligned calculator display
- Button-based input
- Light / Dark mode toggle
- Fixed-size window

---

## AppImage

A prebuilt **Linux AppImage** is available in the releases section.

Download, make it executable, and run:

```bash
chmod +x Calci.AppImage
./Calci.AppImage
